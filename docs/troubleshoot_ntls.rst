Troubleshoot Licensing and NTLS
=====================================

When troubleshooting licensing and :ref:`NTLS <licensing>`, the first step is to retrieve the licensing information and NTLS status. You can do so by sending an API request to NTLS. Necessary actions are then to be undertaken, subject to the response content.
