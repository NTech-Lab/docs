Install FindFace Server
====================================

Several types of installation that we offer will surely make your user experience great. Choose the most convenient one, given your architecture outline:

* The cluster deployment can only be done :ref:`step-by-step <install-server>`.
* The standalone deployment can be done :ref:`step-by-step <install-server>`, from a developer-friendly :ref:`console installer <installer>`, and as a fully pre-configured :ref:`virtual machine image <vm>`.
  
  .. warning::
     For highload projects, installation as a virtual machine is not recommended even in test mode.
